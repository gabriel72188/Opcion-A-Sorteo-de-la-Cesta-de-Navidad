## Requisitos Técnicos

* Node.js (v16 o superior)
* TypeScript
* Jest (para testing)

## Instalación

1.  Clona el repositorio:
    ```bash
    git clone ...
    cd sorteo-cesta-typescript
    ```

2.  Instala las dependencias de desarrollo:
    ```bash
    npm install
    ```
    (Esto instalará `typescript`, `ts-node`, `jest`, `ts-jest` y `@types/jest` definidos en `package.json`).
